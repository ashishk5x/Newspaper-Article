# 📰 Newspaper-Article

A simple newspaper article webpage built using **HTML5** and **CSS3**. This project demonstrates typography, text formatting, and basic hover animations while creating a newspaper-style layout.

## 📸 Preview
<img width="2195" height="1950" alt="Screenshot_21-6-2026_192420_" src="https://github.com/user-attachments/assets/8fe85249-56f6-4f23-ba1b-2ed483059ba5" />


A newspaper-inspired webpage featuring:

* Newspaper title
* Publication date
* Headline and sub-headline
* Author section
* Article paragraphs
* Hover animation with scaling and shadow effects

## ✨ Features

* Semantic HTML structure
* Responsive viewport setup
* Newspaper-style typography
* Uppercase text transformation
* Italic sub-headline
* Paragraph indentation
* Enlarged first letter using the `::first-letter` pseudo-element
* Smooth hover animation using CSS `transform` and `transition`
* Box shadow and rounded borders for a modern newspaper card appearance

## 🛠️ Technologies Used
